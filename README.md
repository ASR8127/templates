# Templates
### Collection of variety of reusable components of HTML, CSS and JavaScript.

What you can make:

- Profile card template
- Reviews card template
- Contact section template
- Theme toggle
- Skills section template
- Hero section template
- Navigation Bar
- Projects Section

<i>... and many more</i>
